Création d'une base de données

Nous allons commencer par créer notre base de données :

  • Cliquez sur Nouvelle base de données. Après avoir choisi un nom pour votre base de données (par exemple "db_livres.db")

  • Cliquez sur la base de données que vous venez de créer.

  • Pour créer une table, cliquez sur l'onglet "Exécuter le SQL". Copiez-collez la requête ci-dessous dans l'onglet "Requête" et exécutez là (l'icône "play")

| |
1
CREATE TABLE LIVRES
2
(id INT, titre TEXT, id_auteur INT, ann_publi INT, note INT);
CREATE TABLE LIVRES
(id INT, titre TEXT, id_auteur INT, ann_publi INT, note INT);
